Download openSUSE - Help on Media Types

The live version of openSUSE 11.0 is now available as a GNOME or KDE CD.

The live system can be used as a productive system or rescue system. You can also use it to just check out how openSUSE 11.0 runs on your computer without touching your hard drive. The Live CDs are available as 32bit versions in US English only and also contain, for the first time, an install option on the desktop.
