Download openSUSE - Help on Media Types

openSUSE is available on CD, DVD or via Network. When delivered as CD ISO image, openSUSE requires a download of approximately 700 MB.
When delivered as a single DVD ISO image, openSUSE requires one download of approximately 4 GB.
When delivered via Network, an initial download of 60MB is required.

Live CDs

The CD version of openSUSE contains only support for English and German. Additional language support is available via the openSUSE online repositories.

  • GNOME Live CD contains the default GNOME installation.
  • KDE Live CD contains the default KDE 4.0 installation.

DVD

  • The DVD contains the language support and all packages, including Non-Open Source Software. The following languages are supported: English, Portuguese, French, Italian, Spanish, German, Chinese (Simpl. & Trad.), Japanese, Russian, Czech, Hungarian, Polish, Finnish, Danish, Swedish, Dutch.
  • The Extra Languages CD contains support for any other language. It is only to be used with DVD installations.

Mini CD

  • The Mini CD conatins a bootable system that allows an installation from a network installation repository. This installation method is only recommended for experienced users. More information is available at Installing openSUSE 11.0 from the Internet.
