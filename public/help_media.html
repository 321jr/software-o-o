Download openSUSE - Help on Media Types

openSUSE is available on CDs, 1 DVD or via Network. When delivered as CD ISO images, openSUSE requires three to five separate downloads of approximately 700 MB each, 2.1 or 3.5 GB in total.
When delivered as a single DVD ISO image, openSUSE requires one download of approximately 4 GB.
When delivered via Network, an initial download of 45MB or 90MB is required.

CD

  • CDs 1-3 are needed in any case. They are sufficient when doing a default installion with KDE or GNOME in one of the following languages: cs, de, en, es, fr, it, and pt_BR.
  • CDs 4-5 are needed
    • when installing in one of the following languages: da, fi, hu, ja, ko, km, nb, nl, pl, and zh.
    • when installing packages beyond the scope of the default installation
  • The Non-Open Source Software CD contains packages with closed source or non-OSI-compliant licenses.
  • The Extra Languages CD contains support for the following languages: af, ar, az, bg, bn, br, bs, ca, cy, el, eo, et, eu, fa, fy, ga, gl, he, hi, hr, is, kk, ko, lt, lv, mk, mn, ms, nds, nn, pa, ro, ru, rw, se, sk, sl, sr, srLatn, ss, sv, ta, tg, tr, uz, vi

DVD

  • The DVD contains the language support and all the packages of the 5 CDs plus the Non-Open Source Software CD.
  • The Live DVD is a complete Linux system that runs entirely from DVD. Installing from the Live DVD is not possible.

Mini CD

  • The Mini CD conatins a bootable system that allows an installation from a network installation repository. This installation method is only recommended for experienced users. More information is available at Installing openSUSE 10.2 from the Internet.
